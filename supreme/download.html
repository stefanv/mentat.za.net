Installation

Use git to clone the source tree:

git clone https://github.com/stefanv/supreme.git
cd supreme

Install system-wide:

  • python setup.py install

Or compile in-place:

  • python setup.py build_ext --inplace

Check that everything works:

$ nosetests supreme
